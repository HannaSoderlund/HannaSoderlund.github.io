SnakeSweeper

Javascript game

I worked as part of a Scrum team to build a custom minesweeper-esque JavaScript game. The user was able to select the size grid to play on, as well as how many ‘mines’ to find. We built functionality to validate user input and then programmatically generate the board with randomised hit boxes. Using event handlers, a modal displayed on hit or miss. We built a ‘lives left’ counter and on finishing the game, the user was presented with the opportunity to play again with a new randomised deck.
