Stressful Todo App

Node.js, Express, MongoDB, HTML, CSS, Git

To get to know Node.JS, Express and MongoDB, I built a simple todo application with a (stressful) Christmas theme. This app lets the user add, mark complete and delete tasks, as well as add custom “tags” to each task. I used the Controller Services pattern for the backend architecture, and really enjoyed using Node with Express for routing. Though this tiny app certainly doesn’t use MongoDB to its full potential, it was a joy to work with and I hope to use it more in the future. With more time I would’ve liked to refactor the frontend using React (and the UX leaves a lot to be desired!).
