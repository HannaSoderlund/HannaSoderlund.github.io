Muppets App

PHP, MySQL, HTML, CSS, Git

OOP project following SOLID principles, we built a PHP application for displaying information about a range of muppets on the front end. We used the Hydrator design pattern to interact with data from a MySQL database, and created functionality to search and generate a random muppet. We built error handling and wrote unit tests where possible. We pair programmed, and code reviewed each others’ work on finishing each task, with the Academy trainer code reviewing each story before merging into main. I particularly enjoyed creating the app architecture and working on the search functionality for the project.
