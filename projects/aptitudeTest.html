Aptitude Test

React, HTML, CSS, Git

For this project, we worked together with two other Academy dev teams on the same code base to rebuild a legacy project previously written in TypeScript. To make the application scalable, we created reusable components working in React. We fetched the data for the test using an existing RESTful API and built a dynamic front end to display the questions and answers using the React-Bootstrap component library and SCSS. To pass variables around our app we used useState() and props, and useEffect() to keep an eye on the score. Once the test was submitted the applicant’s score was calculated and sent as a POST request to update the database.
