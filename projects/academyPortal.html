iO Academy Portal

PHP with Slim, MySQL, JavaScript, HTML, CSS, Git

We worked with two other student development teams on a legacy codebase - the iO Academy portal where academy admins can follow an applicant's journey through application stages. Our Scrum team first worked on fixing two existing bugs, and then built functionality to lock applicant stages to prevent deletion and make the portal safer to use. We also worked on improving the usability of an option select modal, making it clearer to the user where in the process an applicant currently is and what stage is next. The project was built using the PHP framework Slim using the MVC pattern, and used namespacing and autoloading in line with PSR-12. We unit tested with PHPUnit.
