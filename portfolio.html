I am currently studying software development at iO Academy, where I have worked on a range of projects as part of a Scrum team. Below are my favourites to date!

Project 1 - MuppetApp

OOP project following SOLID principles, we built a PHP application for displaying information about a range of muppets on the front end. We used the Hydrator pattern to retrieve data from a MySQL database, and created functionality to search and generate a random muppet. We built error handling and wrote unit tests where possible. I particularly enjoyed creating the app architecture for the project.

Project 3 - NextWeekProject

Working in React, my team rebuilt a legacy project previously written in TypeScript - the aptitude test for new applicants at iO Academy. We fetched the data for the test using an existing RESTful API and built the front end to display the questions and answers using React-Bootstrap. We used the useEffect and useState hooks to keep an eye on the score, and once the test was submitted the applicant’s score was calculated and sent as a POST request to update the database.
